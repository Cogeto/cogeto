# hr-w001 — web stranica: cjenik

Prvi hrvatski web-source slučaj (Priority 5 Part A), autoriran idiomatski, ne
prevođen. `source.txt` je IZLAZ fetchera — čisti tekst fiktivne stranice s
cjenikom (naslov u prvom retku, kako ga web SourceReader dodaje).

Što provjerava: činjenice s iznosima u trećem licu (iznosi se nikad ne
generaliziraju) i datirana promjena cijene ("od 1. listopada 2026.") koja mora
dobiti `valid_from` interval — web činjenica stari pošteno. `source_date` je
vrijeme dohvata: "as of" sidro svih vremenskih izraza na stranici.
