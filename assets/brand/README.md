# Cogeto brand assets

- `cogeto-final-logo-horizontal.svg` — **primary logo** (wordmark, navy + teal); default for light backgrounds.
- `cogeto-final-logo-dark.svg` — logo variant for **dark backgrounds**.
- `cogeto-final-logo-mono.svg` — single-color (navy) variant for print/monochrome contexts.
- `cogeto-final-logo-horizontal-tagline.svg` — primary logo with tagline; marketing surfaces.
- `cogeto-final-logo-stacked.svg` — vertical lockup for square/narrow placements.
- `cogeto-final-icon.svg` — **icon-only mark** (square); avatars, small UI.
- `cogeto-final-favicon.svg` — favicon-optimized icon variant.
- `cogeto-final-icon-tile.svg` — icon on a tile background; app-icon contexts.
- `cogeto-icon-animation.html` / `cogeto-logo-animation.html` — animated demos (reference only).

**Source of truth:** the SVGs, with `cogeto-final-logo-horizontal.svg` as the canonical logo.
